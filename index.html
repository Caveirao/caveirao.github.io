Robô Caveirão

Python 3.6

A parte lógico do robô foi implementada a partir de linguagens como Python e C++ por oferecem maior versatilidade ao projeto.

RaspberryPi 3B

A single-board computer (SBC) Raspberry ficou encarregada de embarcar o software controlador e gerenciar poderosos motores.

Estrutura em Alumínio

Peças exclusivas e de excelente qualidade, à base de aluminio, foram confeccionadas para compor a parte física do robô.

Desenvolvido por discentes do 4º ano de Engenharia da Computação na UniNorte Laureate, o Robô Caveirão é um projeto para a competição de robótica Robô Sumô 2017 de Manaus-Am.
